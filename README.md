# Family Meal Planner 🍽️

A family meal planning app that syncs with Google Calendar and Google Sheets.

## Features
- 📅 Monthly calendar view with meal assignments
- 📆 Infinite scroll week view
- 🍳 Meal list with ratings, cooldown tracking, and smart search
- 🔄 Syncs meals to Google Sheets (shareable database)
- 📱 Adds meals as events to Google Calendar (visible on all devices)
- ⭐ Per-person ratings (Ron, Angela, Ellen, Claire)
- 🏒 Tag system: hockey nights, crockpot meals, Ron-away nights, etc.

## Live App
**https://bigelvis.github.io/meal-planner**

## Setup
No setup needed — just visit the URL above and sign in with Google.

## Search Tips
- `R:5` — Ron's top-rated meals
- `E:4` — Ellen's meals rated 4 or higher  
- `hockey` — meals tagged for hockey nights
- `crockpot` — meals Ron can make in the slow cooker
- `angela only` — Angela's special weekend dishes

## Tech Stack
- Vanilla HTML/CSS/JS (no frameworks)
- Google Sheets API (meal database)
- Google Calendar API (meal scheduling)
- GitHub Pages (hosting)
